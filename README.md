# MQ-FIRE Sensor
### Projeto de sensor de fumaça usando MQ-2 e ESP-32
 O MQ-FIRE é parte do projeto de curricularização da FATEC Cruzeiro, onde temos a missão de desenvolver uma proposta que atenda as necessidades da comunidade de Cruzeiro SP, nosso grupo se uniu a
 AAP(Associação Amando o Próximo) para desenvolver um sistema de detecção de fumaça.
 O MQ-fire é um sistema de segurança para detecção de gases como: GLP(gás liquefeito de petróleo), fumaça, álcool, propano, metano, hidrogênio e monóxido de carbono(CO). Após detecção dos gases o sistema envia alertas ao usuário, através do site e do aplicativo.</p>
O sistema também gera dashboards sobre o volume de gases no ambiente e faz o acompanhamento em tempo real do local monitorado, mostrando o status atual da sala, que pode ser estável, em alerta ou instável, e deixa salvo em uma tabela todos os alertas e instabilidades.</p>
